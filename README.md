# Real-Time Background Verification System

A full-stack web application for secure, real-time background verification between HR and referees.

## Features
- Role-based access (HR / Referee)
- Secure invite-based referee onboarding
- Guided verification questions
- Real-time chat using Socket.IO
- Verification progress tracking
- Persistent message storage (PostgreSQL)
- Responsive UI

## Tech Stack
Frontend:
- React
- Tailwind CSS
- Framer Motion

Backend:
- Node.js
- Express
- Socket.IO
- PostgreSQL
- JWT

## How It Works
1. HR creates a verification room
2. HR sends invite link to referee
3. Referee answers predefined questions
4. HR views structured responses in real time
